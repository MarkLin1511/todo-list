# Habit Tracker

A small vanilla JavaScript habit tracker that lets you add habits, check them off once per day, and track streaks over time.

## Main Site

Open `index.html` in a browser to use the habit tracker.

## Features

- Add custom habits with color tags
- Mark habits complete for the current day
- Track total check-ins and current streaks
- Reset today's progress without deleting the habit
- Persist data with `localStorage`

## Project Structure

- `index.html`, `styles.css`, `app.js`: main habit tracker site
- `habit-tracker/`: standalone copy of the habit tracker files
- `todo-list/`: original to-do list app
